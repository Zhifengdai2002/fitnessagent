# FitnessAgent

A LangGraph-based fitness coaching agent that will generate plans, track feedback, and adapt over time.

## Phase 1 status

The repository currently includes:

- Project skeleton and module boundaries
- Dependency list in `requirements.txt`
- Environment template in `.env.example`
- A minimal Streamlit entry point in `app.py`

## Quick start

1. Create a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Copy `.env.example` to `.env` and fill in your keys.
4. Run `streamlit run app.py`.
